# ecomer-kargazin

IMPORTANTE LEER!!!
 para inicializar correr en la terminal npm start 
 -el proyecto corre en [host local](http://localhost:3000/)
 -para que se vea la web hay que entrar en http://localhost:3000
 
 
 los links no estan funcionando por el momento <--- pendiendes de arreglo
 
 Metodo de prueba
 paso 1- crear usuario haciendo click en el icono de usuario -login-, para neuvos usuarios dirigirse parte inferior y registarce
 paso 2 - crear una cuenta master/seller entrar a http://localhost:3000/seller, aplicar para ese puesto (se peude ingresar cualquier valor)
 paso 3 - automaticamente lo llevara a dashboard donde podra crear sus productos y subirlos a la web
 
 tanto usuarios, como habilitados a seller y productos se guardan en firebase, y las imagenes que se pueden subir van estar almacenadas en AWS
 
 paso 4 -una vez creado el producto se puede modificarlo, guardar en borrador o subirlo a la lista de productos, 
 este ultimo permite llevarlo al carrito de compras y finalizar la transaccion
 
 paso 5 - se puede dar una calificacion producto/servicio por medio del sector review (sera almacenado en firebase)
